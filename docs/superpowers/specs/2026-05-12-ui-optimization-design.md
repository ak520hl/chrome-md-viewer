## 5. 交互改进

### 过渡动画
- 快速: `150ms cubic-bezier(0.4, 0, 0.2, 1)`
- 普通: `250ms cubic-bezier(0.4, 0, 0.2, 1)`
- 慢速: `350ms cubic-bezier(0.4, 0, 0.2, 1)`

### Hover 效果
- 按钮: `translateY(-1px)` + 紫色阴影
- 文件项: 背景色变化 + 轻微左移
- TOC 项: 背景色变化 + 左边框高亮

### 滚动行为
- 平滑滚动: `scroll-behavior: smooth`
- 自定义滚动条: 4-6px 宽度，圆角

---

## 6. 布局优化

### 三栏布局
- 左侧边栏: `280px` (可折叠至 `56px`)
- 内容区: 自适应，最大宽度 `720px`
- 右侧 TOC: `240px`

### 间距系统
| 变量 | 值 | 用途 |
|------|------|------|
| xs | `4px` | 最小间距 |
| sm | `8px` | 小间距 |
| md | `16px` | 中间距 |
| lg | `24px` | 大间距 |
| xl | `32px` | 超大间距 |
| 2xl | `48px` | 特大间距 |
| 3xl | `64px` | 最大间距 |

---

## 7. 具体改进点

### 侧边栏
- 折叠/展开按钮: 更大的点击区域，hover 效果
- Tab 切换: 紫色高亮，平滑过渡
- 文件列表: 悬停时轻微左移，紫色高亮
- 目录树: 层级缩进优化

### 内容区
- 标题: 更大的字号对比，底部边框
- 代码块: 深色背景，圆角
- 引用块: 紫色左边框
- 表格: 斑马纹，hover 效果
- 链接: 紫色，hover 下划线

### 右侧 TOC
- 标题: 大写字母，间距
- 层级缩进: 清晰的视觉层级
- 活跃项: 紫色高亮 + 左边框

---

## 8. 响应式设计

| 断点 | 布局 |
|------|------|
| > 1024px | 三栏布局 |
| 768px - 1024px | 两栏，右侧 TOC 隐藏 |
| < 768px | 单栏，侧边栏可折叠 |
